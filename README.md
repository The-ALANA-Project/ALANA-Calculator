A transparent, NFT-gated internal tool built by [The ALANA Project](https://the-alana-project.xyz) Infrastructure & Strategy team. It lets verified community members submit quests, calculate their $ALANA token rewards based on effort and impact, and have their work reviewed and approved by elected Nucleus Guardians — bringing fair, onchain accountability to community contribution.

## What it does

- **Quest submission** — Contributors submit completed tasks from a shared quest library, input their Battery Life and Quality estimates, and attach proof of work
- **Reward calculation** — A transparent formula based on reach, organisational impact, customer impact, battery life, quality, and branch factor determines the $ALANA reward
- **Guardian review** — Nucleus Guardians review submissions, adjust values if needed, and approve or request changes
- **Community analytics** — A public-facing analytics board shows approved quests, $ALANA distributed, contributor activity, and output by branch over time
- **Template management** — Guardians can create, edit, and publish quest templates via the Guardian Panel

---

## Tech stack

- **Frontend** — React 18, TypeScript, Vite, Tailwind CSS v4
- **Wallet / Auth** — Wagmi, Reown AppKit, Optimism network
- **NFT gating** — Unlock Protocol (on-chain key checks for Guardian, Specialist, Apprentice roles)
- **Backend** — Supabase Edge Functions (Deno + Hono), Supabase KV store
- **Charts** — Recharts
- **Animation** — Motion (Framer Motion v11)
